Cara Menggunakan Web Font Loader Pada Blogger


Konten
  • Pengertian
  • Keuntungan
  • Cara Penggunaan
  • Google Font
  • Google Icons
  • Typekit Font

Web Font Loader gives you added control when using linked fonts via @font-face.
It provides a common interface to loading fonts regardless of the source, then adds a standard set of events you may use to control the loading experience.
The Web Font Loader is able to load fonts from Google Fonts, Typekit, Fonts.com, and Fontdeck, as well as self-hosted web fonts. It is co-developed by Google and Typekit.

Faham ? ini terjemahannya.

Web Font Loader memberi Anda kontrol tambahan saat menggunakan font terhubung melalui @font-face.
Ini menyediakan antarmuka umum untuk memuat font terlepas dari sumbernya, kemudian menambahkan serangkaian peristiwa standar yang dapat Anda gunakan untuk mengontrol pengalaman memuat.
Web Font Loader dapat memuat font dari Google Fonts, Typekit, Fonts.com, dan Fontdeck, serta font web yang dihosting sendiri. Ini dikembangkan bersama oleh Google dan Typekit.

Masih belum mengerti juga ?

Apa itu Web Font Loader ?

Web Font Loader , adalah cara alternatif untuk memuat font dari sumbernya.

Keuntungan Web Font Loader

  • Mengoptimalkan strategi cache.
  • Memanfaatkan penyimpanan cache browser.
  • Mengurangi waktu respons server.

Jika anda adalah pengguna Google Font dan Google Icons, lebih baik gunakan Web Font Loader. Karena sering kali Google Font menjadi masalah Pagespeed.
